Step-by-Step Xiaomi Rooting Process

Step 1: Request Bootloader Unlock Permission

  1. Go to Settings > Additional Settings > Developer Options
  2. Find "Mi Unlock Status" option
  3. Tap "Add account and device"
  4. Log in with your Mi Account
  5. Accept terms and conditions
  6. Submit unlock request
  7. Wait for approval (7+ days typically)
üí° Pro Tip: The waiting period varies by region and device model. Some users report shorter periods, while others may wait longer. Be patient and don't attempt to bypass this officially.

Step 2: Download and Setup Mi Unlock Tool

  1. Download Mi Unlock Tool from official Xiaomi website
  2. Install and run the tool on your computer
  3. Log in with the same Mi Account used on device
  4. Install Xiaomi USB drivers if prompted

Step 3: Unlock Bootloader

  1. Boot device into fastboot mode (Power + Volume Down)
  2. Connect device to computer via USB
  3. Open Mi Unlock Tool and click "Unlock"
  4. Tool will check unlock status and permissions
  5. If approved, click "Unlock Now"
  6. Wait for unlock process to complete
  7. Device will factory reset and reboot

Step 4: Install Custom Recovery (TWRP)

  1. Download TWRP recovery for your specific Xiaomi model
  2. Boot device into fastboot mode again
  3. Flash TWRP recovery using fastboot:
fastboot flash recovery twrp-filename.img
  1. Boot into TWRP recovery (Volume Up + Power)
  2. Allow modifications when prompted

Step 5: Flash Magisk for Root

  1. Download latest Magisk ZIP to device storage
  2. In TWRP, select "Install"
  3. Navigate to Magisk ZIP and select it
  4. Swipe to confirm flash
  5. Reboot system after installation
  6. Install Magisk Manager APK

Troubleshooting Xiaomi Root Issues

Mi Unlock Tool Not Working

  • Ensure you're using the same Mi Account on device and tool
  • Check if waiting period has actually expired
  • Try different USB ports and cables
  • Reinstall Xiaomi USB drivers
  • Use different computer if issues persist

Waiting Period Issues

  • Waiting period is mandatory and cannot be bypassed safely
  • Period may be longer for newer devices or accounts
  • Keep device connected to internet during waiting period
  • Don't factory reset device during waiting period

TWRP Installation Problems

  • Ensure you have correct TWRP for your exact model
  • Some Xiaomi devices need specific TWRP builds
  • Check XDA forums for model-specific TWRP versions
  • Consider using Orange Fox recovery as alternative
